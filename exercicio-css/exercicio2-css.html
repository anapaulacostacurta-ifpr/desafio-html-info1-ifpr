Exercício 2 – Desafio 1: Estilos Inline

Objetivo: Praticar a aplicação de estilos inline em CSS.

Instruções:

  1. Copie o arquivo index.html para um repositório "laboratorio-front-end" em uma nova pasta "desafio-css-inline".
  2. Localize o elemento <h1> já existente.
  3. Utilize o atributo style diretamente dentro da tag <h1> para aplicar as seguintes propriedades:
    • color: azul;
    • font-size: 24px;
  4. Salve o arquivo, publique a página e abra a página no navegador para conferir o resultado.
  5. Você pode usar os comando no terminal:npm i -g http-server e http-server para visualizar como ficou sua página antes de publicar.

Exemplo esperado

Seu código final deve ficar semelhante a este:


                <h1 style="color: blue; font-size: 24px;">
                Bem-vindo ao Meu Blog
                </h1>

✅ Quando recarregar a página, o título “Bem-vindo ao Meu Blog” deverá aparecer em azul com tamanho de fonte 24px.
