Exercício 3 – Estilos Internos

Objetivo: Praticar a aplicação de estilos Internos em CSS.

Instruções:

  1. Copie o arquivo index.html para um repositório "laboratorio-front-end" em uma nova pasta "desafio-css-inline".
  2. No <head> já existente do index.html, adicione uma tag <style></style>.
  3. Crie um seletor para os elementos <body>: para aplicar as seguintes propriedades:
    • color: slategray;
    • font-size: 16px;
  4. Salve o arquivo, publique a página e abra a página no navegador para conferir o resultado.
  5. Você pode usar os comando no terminal:npm i -g http-server e http-server para visualizar como ficou sua página antes de publicar.

✅ Todos os parágrafos ficam no mesmo tom de cinza (slategray) e com tamanho de letra 16px. O título <h1> não é afetado, pois a regra CSS vale apenas para <p>
