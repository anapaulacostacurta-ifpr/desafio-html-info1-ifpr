Exercício 4 – Estilos Externos

Objetivo: Praticar a aplicação de estilos externos em CSS.

Instruções:

  1. Copie o arquivo index.html para um repositório "laboratorio-front-end" em uma nova pasta "desafio-css-externo".
  2. Crie o arquivo styles.css
  3. Defina para o elemento <body>: para aplicar as seguintes propriedades:
    • font-family: Arial, sans-serif;
    • font-size: 14px;
  4. Salve o arquivo, publique a página e abra a página no navegador para conferir o resultado.
  5. Você pode usar os comando no terminal:npm i -g http-server e http-server para visualizar como ficou sua página antes de publicar.

✅ A página terá uma aparência mais uniforme, com Arial 14px como configuração de texto padrão para todo o site.
