Capítulo 5 – Atributos HTML

O que são atributos HTML?

Atributos HTML são informações adicionais que você pode adicionar às tags para configurar ou modificar o comportamento de um elemento. Eles sempre são colocados dentro da tag de abertura e seguem o formato: 

<elemento atributo="valor">

Tipos de atributos

Os atributos podem ser:

  • Com valor: href="...", type="...", src="..."
  • Booleanos: hidden, disabled, checked (presentes ou ausentes)
  • Globais: funcionam em quase todas as tags, como id, class, style

Exemplo prático


                    <button id="meuBotao" class="botao-verde" disabled>
                    Clique aqui
                    </button>

Esse botão tem três atributos: id, class e disabled. Você pode usar o JavaScript para manipulá-los!

Manipulando atributos com JavaScript

  • element.getAttribute("class") → obtém o valor do atributo
  • element.setAttribute("disabled", "") → adiciona ou atualiza o atributo
  • element.removeAttribute("disabled") → remove o atributo

Tabela de Atributos Comuns

Atributo Função Exemplo
href Define o link de um elemento <a> <a href="site.com">
src Define a imagem ou script externo <img src="logo.png">
alt Texto alternativo para imagens <img alt="Logo">
id Identificador único <div id="box">
class Aplica estilos com CSS <p class="aviso">
hidden Oculta o elemento da página <section hidden>
disabled Desativa elementos interativos <input disabled>
placeholder Texto de ajuda dentro de inputs <input placeholder="Digite aqui">

Resumo

  • Os atributos adicionam funcionalidades ou comportamentos aos elementos HTML.
  • Alguns têm valor (como href), outros são booleanos (como ).
  • Podem ser lidos e manipulados com JavaScript usando getAttribute(), setAttribute() e removeAttribute().
