Capítulo 7 – Caracteres Não Imprimíveis em HTML

O que são caracteres não imprimíveis?

Em HTML, alguns símbolos têm um significado especial e não podem ser usados diretamente, pois fazem parte da própria linguagem. Por exemplo, o sinal < é usado para abrir uma tag. Se quisermos exibir esse sinal na tela, precisamos usar uma "entidade HTML".

O que são entidades HTML?

As entidades HTML são códigos especiais usados para representar caracteres reservados ou símbolos que não podem ser digitados diretamente. Elas sempre começam com & e terminam com ;.

Exemplos comuns

Entidade Resultado Uso
&lt;<Sinal de menor que (usado para abrir tags)
&gt;>Sinal de maior que (usado para fechar tags)
&amp;&O próprio símbolo &
&quot;"Aspas duplas
&nbsp; Espaço em branco que não quebra
&copy;©Símbolo de copyright
&euro;€Símbolo do Euro

Quando usar entidades?

  • Quando quiser exibir uma tag HTML no texto (ex: <p>)
  • Quando quiser mostrar símbolos especiais (©, €, etc.)
  • Quando precisar inserir múltiplos espaços seguidos (&nbsp;)

Exemplo prático


                    <p>Use a tag &lt;p&gt; para criar parágrafos.</p>

Resultado exibido na página:

Use a tag <p> para criar parágrafos.

Resumo

  • Alguns símbolos não podem ser usados diretamente no HTML porque têm funções especiais.
  • As entidades HTML permitem exibir esses símbolos na página.
  • Sempre começam com & e terminam com ;.
