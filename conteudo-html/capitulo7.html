O que é o atributo hidden?

O atributo hidden é um atributo booleano do HTML que serve para ocultar elementos da página. Quando aplicado, o conteúdo continua no código, mas não é exibido ao usuário.

Quando usar?

Ele é útil quando se deseja esconder uma informação temporariamente, sem removê-la do HTML. Pode ser ativado ou desativado com JavaScript para exibir mensagens, feedbacks ou partes de interface.

Exemplo em HTML

                    
<p hidden>Este parágrafo está oculto.</p>
<div hidden>
<h3>Título Oculto</h3>
<p>Este conteúdo não será mostrado na página.</p>
</div>

Esse conteúdo está presente no código-fonte, mas não será exibido ao carregar a página.

Mostrando com JavaScript

                    
<button onclick="document.getElementById('mensagem').hidden = false;">
Mostrar mensagem
</button>
<p id="mensagem" hidden>Agora você consegue me ver!</p>

O botão acima usa JavaScript para remover o atributo hidden e exibir o parágrafo.

Explicação:

Elemento Função
<p hidden> Oculta um parágrafo da visualização da página
document.getElementById(...).hidden = false JavaScript que remove o atributo e mostra o conteúdo
<button onclick="..."> Dispara uma ação ao ser clicado

Resumo

  • hidden é um atributo booleano nativo do HTML.
  • Esconde visualmente o elemento, mas mantém sua existência no DOM.
  • É possível reexibir o conteúdo com JavaScript.
