Capítulo 3 – Sintaxe do HTML5

O HTML5 trouxe uma forma mais simples e moderna de escrever código HTML. Neste capítulo, vamos aprender a sintaxe correta para escrever documentos HTML5 de forma limpa, funcional e compatível com todos os navegadores modernos.

🔤 Uso de letras minúsculas

No HTML5, recomenda-se o uso de letras minúsculas para todas as tags e atributos. Isso melhora a legibilidade e segue boas práticas modernas.

✅ Tags não precisam ser fechadas com "/>"

Tags como <br> e <hr> não precisam ser escritas como <br />. O HTML5 aceita apenas <br> de forma válida.

Exemplo:

                    
                        <p>Texto com quebra de linha<br>Nova linha aqui.</p>

📥 Atributos não precisam de aspas se não houver espaços

É permitido omitir as aspas nos atributos que não possuem espaços ou caracteres especiais, mas o uso de aspas é altamente recomendado para evitar erros.

Com aspas (recomendado):

                    
                        <input type="text" name="usuario">

Sem aspas (válido, mas não recomendado):

                    
                        <input type=text name=usuario>

📄 Comparativo: HTML4 vs HTML5

HTML4 HTML5
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"> <!DOCTYPE html>
Uso de <br /> Uso de <br>
Maior rigidez com sintaxe Mais permissivo e simples

✅ Regras Importantes da Sintaxe HTML5

  • Tags devem estar sempre corretamente abertas e fechadas (exceto as autoencerradas).
  • Não usar maiúsculas em tags e atributos.
  • Evitar espaços em branco desnecessários.
  • Inserir sempre o <meta charset="UTF-8"> dentro do <head>.
