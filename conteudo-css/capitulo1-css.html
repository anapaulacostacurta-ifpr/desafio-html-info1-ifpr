🎨 Capítulo 1 – Introdução ao CSS

O CSS (Cascading Style Sheets) define a apresentação das páginas HTML: cores, fontes, espaçamentos, tamanhos e posicionamento. Entender como incluir CSS no seu projeto é o primeiro passo para criar sites organizados e bonitos.

1.2 Incluindo CSS em uma Página Web

Há três formas principais de aplicar CSS: estilos inline, estilos internos e folhas de estilo externas. Abaixo, você verá como e quando usar cada uma.

1.2.1 Estilos Inline

Aplicados diretamente no elemento HTML pelo atributo style. Úteis para ajustes pontuais, porém dificultam manutenção em projetos maiores. 

<p style="color: blue;">Olá, CSS!</p>

1.2.2 Estilos Internos

Definidos no próprio arquivo HTML, dentro da tag <style> no <head>. Indicados para páginas únicas ou protótipos rápidos. 


            <head>
              <style>
                  p {
                    color: red;
                  }
              </style>
            </head>
            <body>
                <p>Olá, CSS!</p>
            </body>

1.2.3 Folhas de Estilo Externas

O CSS fica em um arquivo separado (.css) e é “linkado” no <head>. É a abordagem mais usada em sites profissionais por facilitar a reutilização e a padronização.

index.html

          <head>
          <link rel="stylesheet" href="styles.css">
          </head>
styles.css

          p {
            color: green;
          }

1.2.4 Prós e Contras

  • Inline: rápido e pontual, mas mistura conteúdo e apresentação.
  • Interno: centraliza estilos na página, porém pouco escalável.
  • Externo: organizado e reutilizável, porém exige carregar um arquivo a mais.

Conclusão

Escolha conforme o tamanho e a complexidade do projeto. Para sites com múltiplas páginas, prefira folhas de estilo externas.
