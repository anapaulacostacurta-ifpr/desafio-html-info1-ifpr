🛰️ Passo a Passo: Publicando seu Site com GitHub Pages

Aprenda a publicar seus arquivos HTML usando o GitHub Pages para tornar seu site acessível pela internet.

✅ Pré-requisitos

  • Ter um repositório criado no GitHub (exemplo: laboratorio-tecweb).
  • Ter pelo menos um arquivo index.html no repositório.

🚀 Etapas para publicar

  1. Acesse seu repositório no GitHub: laboratorio-tecweb
  2. Clique em >Settings> (Configurações): no menu superior do repositório.
  3. No menu lateral, clique em Pages: pode estar dentro de “Code and automation”.
  4. Configure o GitHub Pages:
    • Branch: escolha main
    • Folder: selecione /(root)
    Clique em Save.
  5. Aguarde alguns segundos: o GitHub irá gerar o link do seu site automaticamente.
  6. Acesse a URL pública do seu site.
💡 Dica: Sempre use o arquivo index.html como ponto de entrada para que o GitHub Pages reconheça a página inicial.

👍 Parabéns!

Agora você sabe como publicar seus projetos e ver o resultado no navegador, como um site real na web.
