🧭 Como Criar o Repositório no GitHub

Este tutorial orienta você a criar um repositório no GitHub que será usado para armazenar os exercícios e projetos de HTML, CSS e JavaScript desenvolvidos ao longo do curso.

✅ Pré-requisitos

  • Ter uma conta no GitHub
  • Estar logado(a) na sua conta

🚀 Passo a passo

  1. Acesse https://github.com
  2. No canto superior direito e selecione “New repository”
  3. Preencha os campos:
    • Repository name: laboratorio-tecweb
    • Description: Exercícios práticos de HTML, CSS e JavaScript da Disciplina Tecnologia Web - 2026
    • Public: ✅ deixe marcado como público
    • ✔️ Add a README file: Marque esta opção
  4. Clique no botão verde Create repository

Pronto! Agora você tem seu repositório configurado para começar a praticar HTML, CSS e JavaScript de forma organizada.
