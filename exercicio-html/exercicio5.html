Pesquisa, Criação e Revisão de Página HTML

Passo 1 — Escolha do tema

  • Acesse a o link e escolha um tópico novo para você: Formatação.
  • Leia e estude o conteúdo da página.
  • Você pode usar o ChatGPT para tirar dúvidas, pedir explicações alternativas ou exemplos para entender melhor o tema. Por exemplo: “ChatGPT, explique para mim o elemento HTML <b> com um exemplo simples”.
  • Caso você utilize o ChatGPT deverá mencionar nas Referências, conforme abaixo: OPENAI. ChatGPT (versão). Resposta gerada a uma pergunta. [S.l.], 2024. Disponível em: chat.openai.com. Acesso em: [Dia Mes. Ano].

Passo 2 — Montagem de página HTML

- Crie uma página HTML formatacao.html em uma nova pasta chamada atividade-pesquisa no repositorio laboratorio-tecweb que contenha:

  • Um título com o nome do tema escolhido (exemplo: "Elementos HTML <b> HTML").
  • Um parágrafo que explique com suas próprias palavras o que é o tema.
  • Um exemplo simples em HTML que ilustre o tema (pode ser exibido no elemento HTML<pre>).
  • Uma imagem relacionada, se quiser. Como colocar imagem Criar Imagem
  • Pelo menos 2 links relacionados:
    • Link para a página do W3Schools sobre o tema. Como criar link: Criar Link
    • Um outro link complementar (exemplo: vídeo, artigo).

Importante: Use apenas os elementos HTML que você já aprendeu até o momento (títulos, parágrafos, imagens, links, etc.). Não use CSS ou estilização ainda.
