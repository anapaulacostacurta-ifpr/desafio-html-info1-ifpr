Exercício 5 - Pesquisa, Criação e Revisão de Página HTML

Passo 1 — Escolha do tema

  • Acesse a seção HTML do W3Schools: https://www.w3schools.com/html/
  • Escolha um tópico novo para você: Formatação.
  • Leia e estude o conteúdo da página.
  • Você pode usar o ChatGPT para tirar dúvidas, pedir explicações alternativas ou exemplos para entender melhor o tema. Por exemplo: “ChatGPT, explique para mim o elemento HTML <b> com um exemplo simples”.

Passo 2 — Montagem de página HTML

- Crie uma página HTML index.html em uma nova pasta chamada atividade-5-pesquisa.html que contenha:

  • Um título com o nome do tema escolhido (exemplo: "Elementos HTML <b> HTML").
  • Um parágrafo que explique com suas próprias palavras o que é o tema.
  • Um exemplo simples em HTML que ilustre o tema (pode ser exibido no elemento HTML<pre>).
  • Uma imagem relacionada, se quiser.
  • Pelo menos 2 links relacionados:
    • Link para a página do W3Schools sobre o tema,
    • Um outro link complementar (exemplo: vídeo, artigo).

Importante: Use apenas os elementos HTML que você já aprendeu até o momento (títulos, parágrafos, imagens, links, etc.). Não use CSS ou estilização ainda.

Passo 3 — Troca e revisão

- Troque a página com outro aluno.
- A revisão será a próxima o exercício 6.

Dicas para usar o ChatGPT com responsabilidade

  • Use o ChatGPT para entender o conteúdo, tirar dúvidas e pedir exemplos.
  • Não copie respostas diretamente, tente reformular com suas próprias palavras para aprender melhor.
  • Se pedir código, entenda cada linha antes de usar.
  • Use o ChatGPT para revisar seu texto, mas também faça sua própria reflexão crítica.
