Estrutura HTML com UTF-8

Crie um arquivo chamado index.html com a seguinte estrutura HTML básica. Certifique-se de incluir a meta tag para definir a codificação UTF-8, garantindo que os caracteres e acentos sejam exibidos corretamente.

📄 Estrutura Requerida: 

                    
                        <!DOCTYPE html>
                        <html lang="pt-BR">
                        <head>
                        <meta charset="UTF-8">
                        <title>Titulo Página</title>
                        </head>
                        <body>
                        <h1>Titulo Texto</h1>
                        <p>Primeiro Parágrafo</p>
                        </body>
                        </html>

🎯 O que você precisa fazer?

  1. Criar o arquivo index.html na raiz do projeto Minha-Primeira-Pagina no seu repositório do GitHub.
  2. Adicionar o código acima no arquivo "index.html", garantindo que a meta tag <meta charset="UTF-8"> esteja presente.
  3. Fazer o commit e o push do arquivo para o GitHub.
  4. Acessar a página publicada no GitHub Pages para verificar se os caracteres acentuados e especiais estão sendo exibidos corretamente.

✅ Objetivo da atividade:

Entender a importância da meta tag <meta charset="UTF-8"> e como ela garante que o conteúdo seja interpretado corretamente em diferentes idiomas e alfabetos.

💡 Dica:

Se os caracteres aparecerem de forma errada (como "caixinhas" ou símbolos estranhos), verifique se o seu editor de texto e o próprio arquivo estão realmente salvos com a codificação UTF-8.
